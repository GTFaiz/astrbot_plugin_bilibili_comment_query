# astrbot_plugin_bilibili_comment_query
## 简介
AstrBot 插件，通过 **B站 UID** 查询用户近期评论，支持分页查询，自动整理用户曾用名、评论内容、发布时间、点赞回复、视频UP主等信息，**以图片形式优雅渲染返回**。

## 功能特性
- ✅ 通过 `/uid` 指令查询指定 B 站 UID 用户评论
- ✅ 支持自定义页码、每页条数分页查询
- ✅ 自动展示用户**当前昵称 + 历史曾用名**
- ✅ 评论卡片展示：视频标题、评论内容、发布时间、点赞、回复、视频UP主
- ✅ 可自定义 API 地址、请求超时、渲染超时
- ✅ 支持 HTTP 代理配置，适配特殊网络环境
- ✅ 输入参数校验、HTML 特殊字符转义防注入
- ✅ 完善异常处理：超时、网络错误、API 异常、渲染失败友好提示

## 安装方式
1. 将本插件文件夹放入 AstrBot 插件目录 `plugins`
2. 重启 AstrBot 即可自动加载
3. 在 AstrBot 控制台可查看插件配置项并自定义修改

## 使用指令
### 基础格式
```
/uid <B站UID> [页码] [每页条数]
```

### 使用示例
```
/uid 12345678
/uid 12345678 1
/uid 12345678 1 20
```
- 不填页码默认第 1 页
- 不填每页条数使用配置默认值
- 每页条数受配置最大条数限制，防止图片过长

## 配置说明
配置文件由 `_conf_schema.json` 定义，可在后台自定义：

### api 接口配置
- `base_url`：API 基础地址，末尾**不要加斜杠**
- `timeout`：API 请求超时时间（秒）

### page 分页配置
- `default_page_size`：默认每页评论数
- `max_page_size`：最大允许每页条数，限制防止图片过高

### render 图片渲染配置
- `timeout`：HTML 转图片渲染超时（秒）
- `show_all_names`：是否在图片显示用户曾用名

### proxy 代理配置
- 填写如 `http://127.0.0.1:7890` 启用代理
- 留空则不使用代理

## 依赖要求
- Python 3.8+
- astrbot-api 框架环境
- aiohttp 异步网络库
- AstrBot 内置 HTML 转图片(T2I)服务正常可用

## 异常排查
1. 请求超时：检查网络、代理配置、调大 api 超时时间
2. 图片生成失败：检查 AstrBot T2I 服务是否正常，调大 render 渲染超时
3. API 请求失败：检查 base_url 是否正确、接口是否可用
4. 参数报错：确保 UID、页码、条数均为正整数
